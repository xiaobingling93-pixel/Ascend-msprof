## 版本命名规则

MindStudio Profiler 当前版本标签主要采用以下格式：

```text
tag_MindStudio_<major>.<minor>.<patch>.<stage>_<seq>
```

示例：

```text
tag_MindStudio_26.0.0.B090_004
tag_MindStudio_26.0.0.Beta1_001
tag_MindStudio_26.0.0-alpha.1
```

字段说明如下：

| 字段 | 说明 |
|------|------|
| `26.0.0` | 主版本号 |
| `alpha.1` | Alpha 预发布版本 |
| `Beta1_001` | Beta 阶段版本 |
| `B090_004` | 阶段性构建版本，`090` 表示阶段编号，`004` 表示该阶段下的序号 |
